Navigation approach

The app currently has no nav. Which approach fits better for the home dashboard?

A

Persistent top nav bar

A slim header across all pages: Median logo/home link, Search, Profile avatar. Added now and shared everywhere going forward.

Pros

  • Users can always get anywhere
  • Sets up the full app structure now

Cons

  • More work up front
  • Need to add to every page
B

Dashboard-only links

No global nav. The dashboard itself has links to Search and your Profile via widget cards and a small settings link in the corner. Other pages stay as-is.

Pros

  • Faster to build
  • Dashboard is the hub

Cons

  • No way to get home from search or media pages
  • Will need nav later anyway
